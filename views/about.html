Keep track of your daily activities, tasks and goals. Create list to organize your task. You can even track the time you spent on each task.

Register Now! or Log in

Features

Register

Create a new user with your email address, and set your Password. You will need to verify you account, from your email. If you forget your passwor, dont worry, you can recover it anytime!

Authentication

After Registration, you will be automatically authenticated. You can logout and login back with your email and password.

Manage Lists

There is a 'Default' list for your tasks, but you can always create new list, and rename it as many times as you want. 'Secret' lists are possible, and they will not be displayed in your task board until you make them 'Visible'. If you happen to delete a List that already has associated tasks, all those tasks will be automatically moved to the 'Default' list.

Create Tasks and Goals

The core idea of this simple application is to create tasks. Just type the task name on the "Search/Create" box and push the + button. If you want to place the task in an specific list, juthen select the list before clicking the + button. Goals are tasks that can track progress, so you have to set a current value and the end value (goal); for example when you want to track those 12 books you are planning to read this year.

Task board

In the main page you will see all your tasks for the selected list. You can use 'order by' to re-organize the display of your tasks. Tasks with a due date will be displayed using red color font (if they are already due). From here you will be able to 'Close' your tasks, delete, archive, and even update their description or name.

Edit Task

Do you want to add comments to your task, set a due date, or move it to a different list? This is the right place to do it.
